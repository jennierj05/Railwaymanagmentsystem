Description
This Repository Contains the Implementation of Railway Management System Built with MERN. Railway Management System is an application built for users to book trains.
Tech Stack
React - Frontend

Express - Backend/API

MongoDB - Database

Node - Runtime

Mongoose - ORM for Database
